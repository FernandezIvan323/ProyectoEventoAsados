REQUISITO ÚNICO
───────────────
Node.js 18 o superior (gratis)
→ Descargalo en: https://nodejs.org  (elegí "LTS")

Si ya lo tenés instalado, podés saltear este paso.


INICIO RÁPIDO (recomendado)
────────────────────────────
1. Descomprimí el ZIP en cualquier carpeta de tu PC
2. Hacé doble clic en:  INICIAR.bat
3. La primera vez instala dependencias (puede tardar 1-2 min)
4. El navegador se abre solo en http://localhost:5173
5. ¡Listo para usar!

Para cerrar la app: doble clic en DETENER.bat
O simplemente cerrá las ventanas de terminal que se abrieron.


INICIO MANUAL (si el .bat no funciona)
───────────────────────────────────────
Abrí DOS terminales y ejecutá:

  Terminal 1 — Backend:
    cd backend
    npm install
    npx prisma migrate deploy
    npm run dev

  Terminal 2 — Frontend:
    cd frontend
    npm install
    npm run dev

Luego abrí: http://localhost:5173


PRIMERA VEZ
───────────
- La base de datos se crea automáticamente (archivo backend/prisma/dev.db)
- No necesitás internet para usar la app (solo para la instalación inicial)
- Tus datos quedan guardados en tu computadora

PREGUNTAS FRECUENTES
─────────────────────
¿Mis datos están seguros?
  Sí. Todo se guarda localmente en tu PC. Nada se envía a internet.

¿Puedo usarlo en varios dispositivos?
  Por ahora es una app local. Podés copiar la carpeta a otra PC.

¿Cómo hago un respaldo?
  Copiá el archivo: backend/prisma/dev.db
  Ese archivo contiene toda tu base de datos.

¿Funciona en Mac o Linux?
  Sí, pero tenés que usar la instalación manual (los .bat son solo Windows).

El navegador no se abre solo, ¿qué hago?
  Abrí manualmente: http://localhost:5173
